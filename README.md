# Heat Stress Dashboard – South India

This project analyzes heat conditions across South India using Google Earth Engine.

## Objective
To evaluate current heat conditions, compare them with historical averages, and assess human heat stress using Heat Index.

## Tools
- Google Earth Engine
- ERA5-Land Dataset

## Data Used
- temperature_2m
- dewpoint_temperature_2m

## Methodology
1. Current temperature (April–May 2026)
2. Historical baseline (April–May 2015–2020)
3. Temperature anomaly
4. Relative humidity (derived)
5. Heat index (feels-like temperature)

##  Key Insight
Coastal regions exhibit higher heat stress due to elevated humidity levels, despite lower air temperatures compared to inland areas.

##  Live App
https://ledallaadi.users.earthengine.app/view/climateanalysis

##  Limitations
- ~9 km resolution (not city-level precision)
- Model-based data (ERA5)
